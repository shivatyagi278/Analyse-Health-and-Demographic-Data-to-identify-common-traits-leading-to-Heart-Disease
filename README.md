# Healthcare Analytics - EDA & Prediction

## 📌 Project Overview
This project analyzes healthcare and demographic data to identify patterns leading to heart disease using Exploratory Data Analysis (EDA) and basic Machine Learning.

---

## 🚀 Features
- Data Cleaning & Preprocessing
- Correlation Analysis (Heatmap)
- Data Visualization (Seaborn, Matplotlib)
- Risk Factor Identification
- Logistic Regression Model (Prediction)

---

## 📊 Key Insights
- Strong correlation between General Health & Physical Health
- Income impacts overall health conditions
- High BMI and age increase disease risk
- Identified high-risk group:
  > High BMI + Low Income + Poor Health

---

## 🤖 Machine Learning
- Model: Logistic Regression
- Accuracy: ~82%
- Used for predicting heart disease risk
